::: tip
带有搜索按钮的输入框。
:::
### 何时使用
```ts
import { HiSearch } from 'hi-kits'
```

### 基本用法

::: demo
```html
<h-search></h-search>

```
:::

### 自定义配置

::: demo
```html
<h-search placeholder="自定义搜索框提示文案"></h-search>

```
:::
### 参数说明

|参数|说明|类型|可选值|默认值
|:--|:--|:--|:-----|:---
| `placeholder`| 占位符 |  `string` | - | -
