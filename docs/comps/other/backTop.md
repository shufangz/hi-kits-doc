::: tip
返回页面顶部的操作按钮。
:::
### 何时使用
- 当页面内容区域比较长时；
- 当用户需要频繁返回顶部查看相关内容时。
```ts
import { HiBackTop } from 'hi-kits/backTop'
```

### 基本用法

最简单的用法。

::: demo
```html
<h-back-top></h-back-top>
将页面划到最下面，点击右下角的按钮

```
:::
