# Accordion 折叠面板

可以折叠/展开的内容区域。

### 何时使用
- 对复杂区域进行分组和隐藏，保持页面的整洁。
- 手风琴 是一种特殊的折叠面板，只允许单个内容区域展开。
<pre class="language-ts">
import { HiAccordion } from 'hi-kits/accordion'
</pre>

### 基本用法

可以同时展开多个面板，这个例子默认展开了第一个。
